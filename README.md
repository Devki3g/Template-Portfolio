# Portfolio Website

موقع شخصي لعرض الأعمال والمشاريع - تصميم عصري وسريع الاستجابة.

## المميزات

- **تصميم متجاوب** - يعمل بشكل مثالي على جميع الأجهزة (هواتف، أجهزة لوحية، حواسيب)
- **واجهة مستخدم جذابة** - تصميم عصري مع تأثيرات حركية سلسة
- **أداء عالي** - سرعة تحميل ممتازة باستخدام HTML، CSS، JavaScript فقط
- **عرض المشاريع** - عرض الأعمال والمشاريع بشكل منظم
- **نموذج تواصل** - نموذج اتصال تفاعلي
- **تحسين محركات البحث (SEO)** - هيكل HTML محسن لمحركات البحث

## التقنيات المستخدمة

- **HTML5** - هيكل الموقع
- **CSS3** - التصميم والتأثيرات (Flexbox، Grid، Media Queries)
- **JavaScript** - التفاعلات والوظائف الديناميكية
- **Font Awesome** - الأيقونات (اختياري)
- **Google Fonts** - الخطوط الحديثة

## التخصيص

1. تعديل المعلومات الشخصية
   · افتح index.html وقم بتحديث:
     · الاسم والمسمى الوظيفي
     · نبذة تعريفية
     · روابط وسائل التواصل
2. تعديل المشاريع
   · في ملف index.html، ابحث عن قسم المشاريع
   · أضف أو عدل المشاريع حسب رغبتك
3. تعديل الألوان
   · في ملف style.css
   
 ## التحسينات المستقبلية

· إضافة وضع الليل (Dark Mode)
· تحسين أداء الصور
· إضافة تأثيرات حركية أكثر
· دعم متعدد اللغات
· تكامل مع API خارجي لعرض المدونات
